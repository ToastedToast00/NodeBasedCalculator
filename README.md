# NodeBasedCalculator
This repo is dedicated to making a node based calculator similar to that of 
[Blender shader setup](https://docs.blender.org/manual/en/latest/editors/shader_editor.html)
 and 
[Unreal Engine's Node system](https://dev.epicgames.com/documentation/en-us/unreal-engine/nodes-in-unreal-engine).
## How to contribute
* Open VS code or any IDE
* Clone the repo
* Make a new branch
* Create a PR to contribute
